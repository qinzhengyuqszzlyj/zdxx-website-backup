闵行区诸翟学校作业管理制度

2021年9月

作业管理是学校日常管理的重要组成部分，是落实全面育人、推进课程教学改革、提高教育质量、激发学生学习兴趣等工作的有效抓手。根据《关于进一步减轻义务教育阶段学生作业负担校外培训负担的意见》《教育部办公厅关于加强义务教育学校作业管理的通知》《上海市加强义务教育学校作业管理措施》《闵行区关于落实“五项管理”工作的实施方案》的相关精神，我校坚持育人为本、立德树人的目标，建立以校长为第一责任人的工作机制，制定作业管理制度，加强对作业来源、设计、布置、批改、分析、反馈、辅导等全程管理。

一、规范作业来源

1. 各教研组、备课组作业来源于上海市审查通过的相关练习册、教材上的相关习题以及部分备课组团队编制的校本作业。

2. 教师不得使用未经学校及上级主管部门备案审查的学习类APP布置作业，不得要求或组织学生购买未经国家或上海市审查通过的教学资料（教学）。

二、科学设计作业

 教师根据课程标准及相关课程指导意见，在备课环节就要科学设计作业，要体现整体设计、限量设计和单元设计等理念。

1. 教师根据实际学情，通过选编、改编、创编等方式，精准设计、精选符合时代育人要求、体现学校特点、适合学生实际的作业，合理确定作业数量。

2. 教师设计的作业与教学目标保持一致，与当天教学内容紧密联系，与学生基础匹配，符合学生最近发展区需求。

3. 作业设计形式多样，题型类型丰富，鼓励设计分层作业、弹性作业和个性化作业，科学设计探究性作业和实践性作业，探索跨学科综合性作业。切实避免机械、无效作业，严禁重复性、惩罚性作业。

4. 作业难度设计从简到难有梯度，分布合理，有一定弹性。

5. 各学科统一作业本和作业格式基本要求。作业题干表述准确，要求清晰明确，学生能准确读懂。

三、合理布置作业

1. 教师根据学段、学科特点及学生实际需要和完成能力，合理布置书面作业、科学探究、体育锻炼、艺术欣赏、社会和劳动实践等不同类型作业。

2. 教师要及时报备学科作业，班主任、备课组、年级组要统筹各学科日常和假期基础性作业以及弹性作业（含电子作业），确保作业总量在合理范围内。各年级各学科不能布置无效、机械重复和大量抄写的作业，要根据学情鼓励布置分层作业、弹性作业和个性化作业。

3. 教师要提前试做拟布置的作业。布置寒暑假“运动处方”，应在开学两周内组织相应体能监测并将结果纳入体育日常成绩。

4. 教师要及时布置作业，原则上要随堂布置作业。布置作业时要向学生明确作业要求及评价方式，应要求学生规范书写作业，提升学生认真完成作业的积极性。

5. 严控书面作业总量。确保小学一二年级不布置书面家庭作业，可在校内安排适当巩固练习；小学其他年级每天书面作业完成时间平均不超过60分钟；初中每天书面作业完成时间平均不超过90分钟。周末、寒暑假、法定节假日也要控制书面作业时间总量。

6. 严禁给家长布置或变相布置作业，严禁要求家长批改作业。

四、认真批改作业

1. 教师要对布置的学生作业全批全改，不得要求学生自批自改。

2. 作业批改要正确规范。批改作业一律用红笔，批改要准确、及时、细致，批改符号要规范、清楚，批改等第和日期应写明。

3. 作业批改要评语恰当。批改作业要经常写批语，指出存在的问题或解决问题的思路，有针对不同对象的指导和要求。建议用鼓励性的评价来激发学生学习和完成作业的兴趣。

4. 重视学生作业订正，且有订正后的再次批改，对作业错误率高的学生要抓好面批。

5. 教师要有作业批改记录，掌握学生完成作业的数量和质量。学生作业不认真或作业缺交，要耐心教育，缺漏作业要求补做。

6. 杜绝要求家长或学生代为批改作业。

五、有效反馈作业

1. 教师批改作业后要做全面分析，应梳理学生作业中出现的错误、统计错误率，进而思考学生作业错误产生的原因，根据分析进而调整教学内容与方法，反思作业质量。

2. 教师通过作业精准分析学情，采取集体讲评、个别讲解等方式有针对性地及时反馈，共性问题集体讲评，个性问题个别讲解，提升作业评析、反馈的针对性和有效性。鼓励科学利用信息技术手段进行作业分析诊断。

3. 教师评讲、反馈作业要及时，要针对暴露的问题有的放矢地讲评。特别要强化对学习有困难学生的辅导帮扶。

4. 教师使用积极性的评语反馈作业，不得使用侮辱、嘲讽言词反馈作业。

六、重视辅导作业

课外辅导是课堂教学的补充，辅导的重点是学习困难生和优秀生，辅导的目的是使全体学生在原有的基础上都有所提高。

1. 教师要充分利用课堂教学时间和课后服务时间加强作业指导，指导小学生基本在校内完成书面作业，初中学生在校内完成大部分书面作业。

2. 鼓励分层辅导作业。对学习优秀生的辅导要做到有目的、有计划、有针对性，帮助其扩大知识面、培养兴趣爱好、发挥个性特长、提高学习能力和综合素养。对学习困难生的辅导要有组织、有计划的进行，以个别辅导为主，要因材施教，及时补上学生知识上的缺漏。对因事因病缺课的学生，要及时补课。

3. 辅导要讲求实效，要全面考虑学生各科课业负担，不能占用正常的上课时间，不能影响其它学科的教学工作，一般以学生自习和课余时间为主。

七、健全作业管理机制

1. 落实作业控制责任制。实行备课组统筹，任课教师个性化调整，班主任协调，家长监督的制度。班主任统筹班级作业总量，对作业较多的科任老师予以提醒，努力做到各学段作业适度适量。加强年级组、学科组作业统筹协调，合理确定各学科作业比例。

2. 加强作业管理检查与评价。加强对作业来源、设计、布置、批改、分析、反馈、辅导等全过程管理。把作业设计、批改、反馈情况纳入对教师专业素养和教学实绩的考核评价。召开学生座谈会，向家长、学生进行作业管理问卷的不记名调查，进一步了解学科老师的作业管理情况，对优秀的教师和班级进行表彰，对作业管理欠佳的老师和班级予以提醒。多次提醒仍不改进者，进行绩效评价降档处理，对于产生严重不良影响者，根据教职工守则予以必要处罚。

3. 落实教研组作业研究机制。学校鼓励各教研组、备课组根据课程标准及相关课程指导意见，围绕作业选用与设计等主题开展校本研修，继续加强校本作业资源建设，不断完善本学科校本作业体系，增强作业的系统性、针对性。

八、学生作业规范

1． 认真复习，独立思考，按时完成书面、口头等各项作业，并按时上交，严禁抄袭作业。

2． 1-3年级书面作业一律使用铅笔，4-9年级书面作业一律使用黑色水笔，订正用蓝色水笔。文字书写要求字体端正，字迹清楚，不写不规范的简化字和已经简化的繁体字，行款整齐，标点正确。

3． 每次作业应按照各学科作业规范的要求，答题时空二格起笔，转行对齐，两次作业间空五行，留出订正空间。也可以视情况另起一页做下一次的作业。

4． 4-9年级在解题过程中，遇到个别字句写错，用笔画上两笔，表示划去作废，不能用橡皮和修正液。如错误面积较大用直尺划掉再重做。

5． 作业中作图一律使用铅笔，画直线和图形必须用作图工具画，图形大小应适当。

6． 作业中的错误必须及时订正。

7． 作业本按序编号，作业本必须保持整洁完好，不得随意撕页和更换。

8． 作业订正要关注正确率，关注详细的成因分析，关注错题收集和讨论。

9． 作业过程中注意力集中，不吃零食、不分心看电视、上网、听音乐以及频繁中途休息，养成良好的作业习惯。

九、 正确引导家长

1. 学校通过家长学校、家委会、家校互动平台引导家长形成科学育儿观和成才观，切实履行家庭教育主体责任，着力关注孩子自主学习能力，培养孩子良好的作业习惯，与学校形成协同育人合力。

2. 学校向家长提供加强亲子沟通、实施时间管理等专题辅导，正面引导家长根据孩子个性特长、成长规律和发展需求进行学业和生涯规划，学会合理安排孩子业余生活，避免造成孩子过重的校外学习负担。保证小学生每天睡眠时间应达到10小时，初中生应达到9小时。

3. 督促孩子回家后主动完成学校布置的作业，引导孩子从事力所能及的家务劳动，激励孩子坚持进行感兴趣的体育锻炼和社会实践，不额外布置其他家庭作业。

4. 通过家长会、家校互动平台、学校公众号等为家长的有效做法提供交流分享的平台，通过智慧传递加强有效经验的传播和再发展。
